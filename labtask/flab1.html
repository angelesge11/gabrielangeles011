Finals Lab Task 1

Encapsulation - A Car That Works

Description

This finals activity shows how encapsulation works in Python by using private attributes and methods in a Car class and displaying object details through a constructor.

Problem

For this program, you are required to define a class named Car with specific properties, methods, and a constructor. The class should have the following:

  • color (type: str) – represents the car’s color
  • price (type: float) – holds the car’s price
  • size (type: str) – indicates the size of the car ('S' = small, 'M' = medium, 'L' = large)

The constructor __init__(self, color: str, price: float, size: str) initializes these properties, with the size value automatically converted to uppercase using size.upper().

The class should include getter methods (get_color, get_price, and get_size) that return the respective property values.

It should also have setter methods (set_color, set_price, and set_size) to modify each property, ensuring that the size value is one of 'S', 'M', or 'L', and any lowercase input is converted to uppercase.

Lastly, the __str__ method should return a formatted string showing the car’s details in the format:
"Car (color) - P(price, formatted to two decimal places) - (size descriptor)"

The size descriptor corresponds to 'small', 'medium', or 'large' based on the size property.

For example:
Car (red) - P19999.85 - medium
Car (blue) - P50000.00 - large
Car (green) - P22000.00 - small

Revised Code:

class Car:
    def __init__(self, model: str, price: float, transmission: str):
        self.model = model
        self.price = price
        self.transmission = transmission.upper()
								 
    def get_model(self) -> str:
        return self.model
    def get_price(self) -> float:
        return self.price
    def get_size(self) -> str:
        return self.transmission
    def set_model(self, model: str) -> None:
        self.model = model
    def set_price(self, price: float) -> None:
        self.price = price

    def set_transmission(self, transmission: str) -> None:
        self.transmission = transmission.upper()

    def __str__(self) -> str:
        transmission_descriptor = {
            'M': 'Manual',
            'A': 'Automatic',
            'H': 'Hybrid'
        }
        return f"Car ({self.model}) - ${self.price:.2f} - {transmission_descriptor.get(self.transmission, 'unknown')}"

car1 = Car("Mazda", 112999.99, "M")
print(car1)

car2 = Car("Toyota", 120000.00, "A")
print(car2)

car3 = Car("Ducatti", 123456.78, "H")
print(car3)
